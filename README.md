# 林野摄影作品集

面向人像摄影求职与商业合作的双语个人网站，包含精选作品、分类图库、摄影师介绍、履历、摄影笔记、联系方式与简历下载。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

## 主要内容

- 三套明亮主题与中英文切换
- Kodak Ultramax 400 风格横向胶片作品展示
- 人像、跟拍与商业作品筛选
- 响应式导航、键盘操作与减少动态效果支持

网站主要内容位于 `app/portfolio.tsx`，视觉样式位于 `app/globals.css`。替换示例照片时，修改 `images` 数组中的图片地址、分类、名称与年份即可。

## 构建

```bash
npm run build
```
